グローバル YouTube OAuth 設定

一般ユーザー向けの「かんたん連携」で使用されるシステムのデフォルトOAuth認証情報です。

⚠️ 注意: ここで設定する Client ID と Client Secret はサーバー全体のデフォルトとして使用されます。 一般ユーザーが「かんたん連携」を実行した際、この認証情報を用いて認証が行われます。

🔑 本物のグローバル OAuth 認証キーを取得する手順（Google Cloud）
  1. Google Cloud Console にアクセス:
    Google Cloud Console にログインし、新しいプロジェクトを作成または既存のプロジェクトを選択します。
  2. YouTube API を有効化:
    「APIとサービス」 > 「ライブラリ」を開き、検索窓に「YouTube Data API v3」と入力して、これを有効化します。
  3. OAuth 同意画面の設定:
    「APIとサービス」 > 「OAuth 同意画面」を開き、User Type を「外部」に設定し、アプリ名や連絡先メールアドレスなどの必須項目を入力します。
  4. 認証情報の作成 (OAuth クライアント ID):
    「APIとサービス」 > 「認証情報」を開き、「認証情報を作成」 > 「OAuth クライアント ID」をクリックします。
  5. アプリケーションタイプとリダイレクト URI:
    アプリケーションの種類を「ウェブ アプリケーション」に選択し、「承認済みのリダイレクト URI」に以下のURLを追加します： https://kimidori-movie-auto-84265380422.asia-northeast1.run.app/api/auth/youtube/callback
  6. キーの取得と設定:
    作成完了後に表示される「クライアント ID」と「クライアント シークレット」をコピーし、上のフォームに貼り付けて保存してください。
